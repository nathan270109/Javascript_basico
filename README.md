## Aula 2 — Variáveis e Constantes

### Conteúdo Programático
- Declaração com `let` e `const`
- Escopo de bloco
- Tipos primitivos: `number`, `string`, `boolean`, `undefined`, `null`
- Operador `typeof`
- Template strings (interpolação)
- Boas práticas de nomeação

### Objetivos
- Compreender a diferença entre `let` e `const`
- Declarar e inicializar variáveis corretamente
- Identificar os tipos de dados primitivos com `typeof`
- Utilizar template strings para montar mensagens dinâmicas

### Ementa
A aula introduz os conceitos fundamentais de armazenamento de dados em JavaScript. O aluno aprende a criar variáveis mutáveis e constantes, entende escopo de bloco e reconhece os tipos primitivos mais comuns. Exercícios práticos envolvem declaração, reatribuição e exibição de valores no console.

## Aula 3 — Operadores

### Conteúdo Programático
- Operadores aritméticos: `+`, `-`, `*`, `/`, `%`
- Operadores relacionais: `>`, `<`, `>=`, `<=`, `==`, `===`, `!=`
- Operadores lógicos: `&&`, `||`, `!`
- Precedência de operadores
- Operador ternário `? :`
- Incremento e decremento: `++`, `--`

### Objetivos
- Realizar cálculos usando operadores aritméticos
- Comparar valores com operadores relacionais e estritos
- Combinar condições com operadores lógicos
- Utilizar o operador ternário para decisões simples

### Ementa
Operadores são as ferramentas para manipular dados. A aula aborda desde a aritmética básica até expressões lógicas complexas, passando pela compreensão da precedência. Exemplos práticos incluem cálculo de médias, verificação de paridade e validação de entradas.

**Atividades desenvolvidas para o Projeto @arizeviecelli**

## Aula 4 — Estruturas de Decisão

### Conteúdo Programático
- `if / else if / else`
- `switch / case`
- Operador ternário como substituto de `if` simples
- Aninhamento de estruturas
- Valores *truthy* e *falsy*
- Boas práticas de legibilidade

### Objetivos
- Implementar fluxos condicionais com `if/else`
- Utilizar `switch` para múltiplos casos
- Aplicar o operador ternário em situações adequadas
- Avaliar condições baseadas em valores *truthy/falsy*

### Ementa
A tomada de decisão é essencial em qualquer programa. A aula ensina as três formas de controle condicional – if/else, switch e ternário – com ênfase na clareza do código. Os alunos resolvem problemas como classificação de notas, dias da semana e validação de senhas.

**Atividades desenvolvidas para o Projeto @arizeviecelli**

## Aula 5 — Laços de Repetição

### Conteúdo Programático
- `for` (contador)
- `while` (condicional)
- `do...while` (execução garantida)
- Comandos `break` e `continue`
- Laços aninhados
- Iteração sobre arrays com `for` clássico
- Introdução a `for...in` e `for...of`

### Objetivos
- Construir loops com `for`, `while` e `do...while`
- Controlar o fluxo com `break` e `continue`
- Criar laços aninhados para problemas matriciais
- Iterar sobre arrays de forma eficiente

### Ementa
Repetição é o que automatiza tarefas. A aula explora os três tipos de laços e seus usos típicos. Exercícios vão de contagens simples até a geração de tabuadas e a busca de números primos. O aluno também aprende a evitar loops infinitos.

**Atividades desenvolvidas para o Projeto @arizeviecelli**

## Aula 6 — Arrays

### Conteúdo Programático
- Criação e acesso a elementos por índice
- Propriedade `length`
- Métodos: `push`, `pop`, `shift`, `unshift`
- `slice`, `splice`, `concat`
- `forEach` (iteração)
- `map`, `filter`, `reduce`
- `indexOf`, `includes`
- Arrays multidimensionais

### Objetivos
- Criar e manipular arrays com métodos essenciais
- Utilizar `forEach` para percorrer listas
- Aplicar `map`, `filter` e `reduce` para transformação, filtragem e acumulação

### Ementa
Arrays são a estrutura de dados mais versátil em JavaScript. A aula cobre desde criação básica até o uso de métodos funcionais. Os alunos praticam ordenação, cópia, remoção e busca, além de trabalharem com matrizes bidimensionais.

## Aula 7 — Objetos

### Conteúdo Programático
- Criação de objetos literais
- Notação de ponto e colchetes
- Adição e remoção de propriedades
- Métodos dentro de objetos (`this`)
- Objetos aninhados
- Iteração com `for...in`
- `Object.keys`, `Object.values`, `Object.entries`
- `JSON.stringify` e `JSON.parse`

### Objetivos
- Construir e acessar objetos com propriedades e métodos
- Navegar por estruturas aninhadas
- Serializar e desserializar dados com JSON

### Ementa
Objetos modelam entidades do mundo real. A aula ensina a criar, modificar e percorrer objetos, além de introduzir o conceito de `this`. Os alunos desenvolvem um cadastro de produtos e um perfil de usuário com endereço.

## Aula 8 — Funções

### Conteúdo Programático
- Declaração de função (function statement)
- Expressão de função (function expression)
- Arrow functions
- Parâmetros e argumentos
- Parâmetros padrão e rest (`...`)
- Escopo e closure
- Callbacks

### Objetivos
- Definir e invocar funções de diferentes formas
- Compreender escopo e closure
- Utilizar callbacks em operações assíncronas

### Ementa
Funções são o coração da reutilização de código. A aula cobre desde a sintaxe básica até arrow functions e closures. Os alunos criam funções puras, funções que retornam funções e usam callbacks em cenários simulados.

## Aula 9 — Manipulação do DOM

### Conteúdo Programático
- Seleção de elementos: `getElementById`, `querySelector`
- Alteração de conteúdo: `innerHTML`, `textContent`
- Manipulação de classes: `classList.add`, `remove`, `toggle`
- Eventos: `addEventListener`, tipos comuns (click, input, submit)
- Criação e remoção de elementos: `createElement`, `appendChild`, `remove`
- Delegação de eventos
- Projeto final: lista de tarefas interativa

### Objetivos
- Selecionar e modificar elementos do DOM
- Responder a eventos do usuário
- Construir uma aplicação simples com JavaScript puro

### Ementa
A aula final une tudo o que foi aprendido para criar páginas dinâmicas. O aluno aprende a manipular o DOM, reagir a cliques e teclas, e construir um mini projeto funcional – como um cronômetro, uma galeria ou uma lista de tarefas.
