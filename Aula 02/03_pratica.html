Passo 1 - Declare três variáveis com let e atribua valores

Crie três variáveis usando let. Uma com seu nome (string), uma com sua idade (number) e uma com seu hobbie favorito (string). Exiba cada uma no console.
Dica: Use console.log() para ver o valor de cada variável.
let nome = "Ana";
let idade = 22;
let hobbie = "ler";
console.log(nome);
console.log(idade);
console.log(hobbie);

Passo 2 - Crie uma constante para PI

Crie uma constante chamada PI com o valor 3.14159. Depois crie outra variável com o raio de um círculo e calcule a área (PI * raio * raio). Exiba o resultado.
Dica: const não pode ser reatribuída.
const PI = 3.14159;
let raio = 5;
let area = PI * raio * raio;
console.log(area);

Passo 3 - Use typeof para cada tipo primitivo

Crie uma variável de cada tipo primitivo e use typeof para descobrir o tipo de cada uma.
Dica: typeof variavel retorna o tipo como string.
let nome = "João";
let idade = 30;
let ativo = true;
console.log(typeof nome);
console.log(typeof idade);
console.log(typeof ativo);

Passo 4 - Converta string para número

Crie uma string com um número dentro e converta para número usando Number().
Dica: Number() converte string para número.
let stringNumero = "50";
let numero = Number(stringNumero);
let resultado = numero + 10;
console.log(resultado);

Passo 5 - Teste escopo com bloco de código

Crie uma variável let fora de um bloco { }. Dentro do bloco, crie outra com o mesmo nome mas valor diferente. Exiba os valores.
Dica: Variáveis let dentro de { } só existem dentro daquele bloco.
let mensagem = "Fora";
{ let mensagem = "Dentro";
console.log(mensagem);
} console.log(mensagem);

Passo 6 - Renomeie variáveis seguindo padrão camelCase

Renomeie variáveis com nomes ruins para camelCase descritivos.
Dica: camelCase: primeira minúscula, próximas maiúsculas.
let nome = "João";
let idade = 25;
let cidade = "SP";
console.log(nome, idade, cidade);

Todos os passos concluídos!
