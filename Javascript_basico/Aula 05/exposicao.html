Laço for: estrutura e uso

O laço for repete um bloco de código um número definido de vezes.

Sintaxe: for (inicio; condicao; incremento) { }

for (let i = 1; i <= 5; i++) { console.log(i); } // 1, 2, 3, 4, 5

Laço while: quando usar

while repete enquanto uma condição for verdadeira. Ideal quando não sabemos quantas repetições serão necessárias.

let contador = 0; while (contador < 5) { console.log(contador); contador++; }

Laço do...while: execução garantida

do...while executa o bloco pelo menos uma vez, depois verifica a condição.

Diferença: while testa antes, do...while testa depois.

let i = 0; do { console.log(i); i++; } while (i < 5);

Controle de fluxo: break e continue

break interrompe o loop imediatamente. continue pula para a próxima iteração.

Exemplo com break:
for (let i = 0; i < 10; i++) { if (i === 5) break; console.log(i); } // 0,1,2,3,4
Exemplo com continue (apenas ímpares):
for (let i = 0; i < 10; i++) { if (i % 2 === 0) continue; console.log(i); }

Laços aninhados

Um laço dentro de outro. Útil para percorrer tabelas e matrizes.

O laço externo executa uma vez para cada iteração completa do interno.

for (let i = 1; i <= 3; i++) { for (let j = 1; j <= 3; j++) { console.log(i + "," + j); } }

Iteração sobre arrays

Podemos percorrer arrays com for usando o índice. O length do array define o limite.

let frutas = ["maçã", "banana", "uva"]; for (let i = 0; i < frutas.length; i++) { console.log(frutas[i]); }

Laço for...in e for...of (introdução)

for...in percorre propriedades de objetos (ou índices de array).

for...of percorre valores de objetos iteráveis (arrays, strings).

Exemplo com for...of (mais simples):
for (let fruta of frutas) { console.log(fruta); }

Evitando loops infinitos

Loop infinito acontece quando a condição nunca se torna falsa. Travam o navegador!

Causas comuns:
  • Esquecer de incrementar a variável de controle
  • Condição sempre verdadeira
  • Usar = em vez de === na comparação

Sempre verifique se a variável de controle está sendo atualizada.
