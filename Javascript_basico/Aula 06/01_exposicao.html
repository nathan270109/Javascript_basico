O que são Arrays?

Array é uma lista ordenada que armazena múltiplos valores em uma única variável. Cada item ocupa uma posição (índice) começando em 0.

let frutas = ["maçã", "banana", "uva"]; console.log(frutas[0]); // maçã

Índices e length

Cada elemento tem um índice numérico. O primeiro é 0, o último é length - 1. length retorna o tamanho do array.

let numeros = [10, 20, 30]; console.log(numeros.length); // 3 console.log(numeros[numeros.length - 1]); // 30

Métodos push, pop, shift, unshift

push() adiciona no final, pop() remove do final, shift() remove do início, unshift() adiciona no início.

let cores = ["azul"]; cores.push("verde"); // ["azul", "verde"] cores.pop(); // ["azul"]

Métodos slice, splice, concat

slice() copia parte do array sem modificar original, splice() adiciona/remove em qualquer posição, concat() junta arrays.

let a = [1,2,3]; let b = [4,5]; let c = a.concat(b); // [1,2,3,4,5]

Iteração com forEach

forEach() executa uma função para cada elemento do array. É a forma mais elegante de percorrer listas.

let numeros = [1, 2, 3]; numeros.forEach(function(n) { console.log(n * 2); }); // 2, 4, 6

Métodos map, filter, reduce

map() transforma cada elemento, filter() seleciona elementos que atendem condição, reduce() reduz array a um único valor.

let nums = [1,2,3,4]; let pares = nums.filter(n => n % 2 === 0); // [2,4]

Busca com indexOf e includes

indexOf() retorna o índice do elemento ou -1 se não existir. includes() retorna true/false se o elemento existe.

let frutas = ["maçã", "banana"]; console.log(frutas.includes("maçã")); // true

Arrays Multidimensionais

Array dentro de array - como uma planilha com linhas e colunas. Útil para matrizes e tabelas.

let matriz = [[1,2], [3,4]]; console.log(matriz[0][1]); // 2
