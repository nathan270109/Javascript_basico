O que são Variáveis?

Variável é um espaço na memória que guarda um valor. Imagine uma caixa com uma etiqueta — a etiqueta é o nome da variável, o conteúdo é o valor.

let nome = "Sora"; // criamos a caixa "nome" e guardamos "Sora" dentro

Cada variável tem um nome único e um valor que pode ser alterado durante a execução do programa.

Declaração com let

Usamos let para criar variáveis que podem ter seu valor alterado.

let idade = 16;
idade = 17; // o valor mudou!

let é a forma moderna e segura de declarar variáveis no JavaScript. Permite reatribuição, mas não redeclaração no mesmo escopo.

Declaração com const

Usamos const para criar constantes — valores que não mudam. Depois de declarada, não pode ser reatribuída.

const PI = 3.14159;
PI = 3; // Erro! Constante não pode ser reatribuída
Use const sempre que o valor não precisar mudar. É uma boa prática.

Tipos Primitivos: string, number, boolean

JavaScript tem 3 tipos primitivos principais:

  • string — texto, entre aspas. Ex: let nome = "Ana";
  • number — números, inteiros ou decimais. Ex: let idade = 25;
  • boolean — verdadeiro ou falso. Ex: let estudante = true;

Esses são os blocos fundamentais para construir dados.

typeof para verificar tipos

O operador typeof revela o tipo de um valor. Útil para depuração e para entender que tipo de dado você está manipulando.

typeof "Olá" → "string"
typeof 42 → "number"
typeof true → "boolean"
Experimente no console do navegador!

Conversão de tipos

Às vezes precisamos converter um tipo em outro.

Number("123") → 123
String(456) → "456"

Cuidado: "10" + 5 = "105" (concatena), Number("10") + 5 = 15 (soma).

Escopo e boas práticas de nomeação

Escopo é onde a variável existe. let e const têm escopo de bloco {}.

Nomes devem ser descritivos e seguir camelCase (primeira letra minúscula, próximas palavras maiúsculas).

let nomeCompleto;
let idadeDoUsuario;

Evite nomes genéricos como x, valor, temp.

Diferenças entre var, let e const

  • var: escopo global/função, pode ser redeclarada (antiga, evitar).
  • let: escopo de bloco, pode ser reatribuída (moderna, usar).
  • const: escopo de bloco, NÃO pode ser reatribuída (moderna, usar).
Regra prática: prefira const, use let quando precisar reatribuir, nunca use var.
