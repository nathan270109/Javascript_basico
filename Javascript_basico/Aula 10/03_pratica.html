Passo 1 - Defina o tema do projeto

▾

Instrução: Escolha um tema para seu projeto final. Pode ser: lista de tarefas, diário pessoal, catálogo de filmes, controle de gastos. Escreva o nome do projeto e as funcionalidades principais no console.

Dica: Planeje antes de codificar. O que seu app vai fazer?

console.log("Projeto: Lista de Tarefas");
console.log("Funcionalidades: adicionar, remover, filtrar");

Passo 3 - Implemente funcionalidade de adicionar item

▾

Instrução: Crie uma função adicionarItem(texto) que adiciona um item a um array e exibe a lista atualizada no console.

Dica: array.push() adiciona no final.

let itens = []; function adicionarItem(texto) { itens.push(texto); console.log(itens); } adicionarItem("Estudar");

Passo 4 - Use localStorage para persistir dados

▾

Instrução: Salve o array de itens no localStorage usando JSON.stringify e depois recupere com JSON.parse.

Dica: localStorage só guarda strings. Use JSON.stringify para converter.

let dados = ["item1", "item2"]; localStorage.setItem("meusDados", JSON.stringify(dados)); let recuperados = JSON.parse(localStorage.getItem("meusDados")); console.log(recuperados);

Passo 5 - Adicione filtro de itens

▾

Instrução: Crie uma função filtrarItens(termo) que recebe um termo e retorna apenas os itens que contêm esse termo (use filter).

Dica: filter com item.includes(termo).

let itens = ["Estudar JS", "Praticar Python", "Ler livro"]; function filtrarItens(termo) { return itens.filter(item => item.includes(termo)); } console.log(filtrarItens("Est"));

Passo 6 - Publique online

▾

Instrução: Crie uma mensagem no console com as instruções para publicar no GitHub Pages: criar repositório, fazer upload, ativar Pages.

Dica: GitHub Pages é gratuito para sites estáticos.

console.log("1. Crie um repositório no GitHub"); console.log("2. Faça upload dos arquivos (index.html, style.css, script.js)"); console.log("3. Vá em Settings > Pages > Source: main branch"); console.log("4. Seu site estará online em https://seuusuario.github.io/seurepositorio/");
