Operadores Aritméticos (+, -, *, /, %)

Os operadores aritméticos realizam cálculos matemáticos. São eles: + (soma), - (subtração), * (multiplicação), / (divisão) e % (resto da divisão, também chamado de módulo).

console.log(10 + 5); // 15 console.log(10 % 3); // 1 (resto)

Operadores Relacionais (>, <, >=, <=, ==, ===, !=)

Operadores relacionais comparam valores e retornam true ou false. > maior, < menor, >= maior ou igual, <= menor ou igual, == igual (valor), === igual (valor e tipo), != diferente.

console.log(10 > 5); // true console.log(10 === "10"); // false

Operadores Lógicos (&&, ||, !)

Operadores lógicos combinam condições: && (AND - todas verdadeiras), || (OR - pelo menos uma verdadeira), ! (NOT - inverte o resultado).

let idade = 20; let ingresso = true; console.log(idade >= 18 && ingresso); // true

Precedência de Operadores

Assim como na matemática, operadores têm ordem de prioridade: 1º parênteses (), 2º multiplicação/divisão/resto (* / %), 3º soma/subtração (+ -), 4º relacionais, 5º lógicos.

console.log(5 + 3 * 2); // 11 console.log((5 + 3) * 2); // 16

Operadores de Atribuição

Atribuem valores a variáveis de forma simplificada: = (atribuição simples), += (soma e atribui), -= (subtrai e atribui), *= (multiplica e atribui), /= (divide e atribui), %= (resto e atribui).

let x = 10; x += 5; // x = 15 console.log(x);

Operador Ternário

É uma forma compacta de decisão: condicao ? valorSeVerdadeiro : valorSeFalso. Útil para decisões simples em uma linha.

let idade = 18; let resultado = idade >= 18 ? "Maior" : "Menor"; console.log(resultado); // "Maior"

Incremento e Decremento

Operadores que aumentam ou diminuem valores em 1: ++ incremento, -- decremento. ++x pré-incremento (incrementa antes de usar), x++ pós-incremento (usa antes de incrementar).

let contador = 0; contador++; // 1 console.log(contador);

Exercícios de Expressões

Hora de praticar! Combine os operadores que aprendeu para resolver problemas: calcule médias, compare valores, combine condições. Use o console para testar suas expressões. Experimente diferentes combinações!
