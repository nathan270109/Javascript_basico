Passo 1 - Calcule a média de três notas

▼

Instrução: Crie três variáveis com notas (number) e calcule a média aritmética delas. Exiba o resultado no console.

💡 Dica: Média = (nota1 + nota2 + nota3) / 3
let n1 = 8;
let n2 = 7;
let n3 = 9;
let media = (n1 + n2 + n3) / 3;
console.log(media);

Passo 2 - Verifique se um número é par ou ímpar

▼

Instrução: Crie uma variável com um número qualquer. Use o operador % (resto) para verificar se o número é par ou ímpar. Se resto for 0, é par.

💡 Dica: numero % 2 retorna 0 para números pares.
let numero = 7;
let resto = numero % 2;
let resultado = resto === 0;
console.log(resultado);

Passo 3 - Combine duas condições com &&

▼

Instrução: Crie duas variáveis booleanas (ex: maior de idade e tem ingresso). Use && para verificar se ambas são verdadeiras. Exiba o resultado.

💡 Dica: && retorna true apenas se TODAS as condições forem verdadeiras.
let maiorIdade = true;
let temIngresso = true;
let podeEntrar = maiorIdade && temIngresso;
console.log(podeEntrar);

Passo 4 - Use || para verificar se uma variável é válida

▼

Instrução: Crie uma variável que pode ser null ou undefined. Use || para verificar se ela tem um valor válido. Se for null, use um valor padrão.

💡 Dica: || retorna o primeiro valor verdadeiro. nome || "Visitante" retorna "Visitante" se nome for vazio.
let nome = "";
let usuario = nome || "Visitante";
console.log(usuario);

Passo 5 - Escreva expressão com operador ternário

▼

Instrução: Crie uma variável com uma temperatura. Use o operador ternário para verificar se está quente (> 25) ou frio. Exiba o resultado.

💡 Dica: condicao ? valorSeVerdadeiro : valorSeFalso
let temperatura = 30;
let clima = temperatura > 25 ? "Quente" : "Frio"; console.log(clima);

Passo 6 - Analise precedência em uma expressão complexa

▼

Instrução: Crie uma expressão que combine +, * e (). Compare o resultado com e sem parênteses para entender a precedência. Exiba ambos.

💡 Dica: Parênteses têm maior prioridade que multiplicação, que tem prioridade que soma.
let semParenteses = 5 + 3 * 2;
let comParenteses = (5 + 3) * 2;
console.log(semParenteses, comParenteses);
