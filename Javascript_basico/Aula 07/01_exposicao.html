O que são Objetos?

Objeto é uma coleção de dados e funcionalidades agrupados. Imagine uma ficha de cadastro: nome, idade, cidade. No JS, criamos objetos com { }.

let pessoa = { nome: "Ana", idade: 25, cidade: "SP" };
Propriedades e valores

Propriedades são os pares chave: valor dentro do objeto. A chave é como uma etiqueta, o valor é o dado armazenado. Podem ser string, number, boolean, array, até outro objeto.

let carro = { marca: "Toyota", ano: 2022, eletrico: false };
Acesso com notação de ponto e colchetes

Podemos acessar propriedades de duas formas: pessoa.nome (notação de ponto) ou pessoa["nome"] (notação de colchetes). Colchetes são úteis quando o nome da propriedade está em uma variável.

let chave = "nome"; console.log(pessoa[chave]);
Métodos em objetos

Métodos são funções dentro de objetos. Eles podem acessar outras propriedades do objeto usando a palavra-chave this.

let pessoa = { nome: "João", saudacao: function() { console.log("Olá, " + this.nome); } };
Objetos aninhados

Um objeto pode conter outro objeto como propriedade. Isso permite criar estruturas de dados complexas e organizadas.

let usuario = { nome: "Ana", endereco: { rua: "ABC", numero: 123 } }; console.log(usuario.endereco.rua);
Iteração com for...in

for...in percorre todas as propriedades de um objeto. A cada iteração, a variável recebe o nome de uma chave.

for (let chave in pessoa) { console.log(chave + ": " + pessoa[chave]); }
Object.keys, values, entries

Object.keys() retorna um array com as chaves, Object.values() retorna os valores, Object.entries() retorna pares [chave, valor].

console.log(Object.keys(pessoa)); // ["nome", "idade"]
JSON.stringify e JSON.parse

JSON é o formato de texto para troca de dados. JSON.stringify() converte objeto em string JSON. JSON.parse() converte string JSON de volta em objeto.

let json = JSON.stringify(pessoa); let obj = JSON.parse(json);
