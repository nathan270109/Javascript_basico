Você é um Robô em uma missão de cadastro de dados. Precisa organizar informações em objetos para criar um banco de dados do robô. Complete os 4 requisitos para ativar o módulo de cadastro.

📝 Requisitos

  • Criar objeto robô: Crie um objeto chamado robo com nome, nivel e energia
  • Acessar propriedades: Use notação de ponto e colchetes para exibir nome e nivel
  • Objeto aninhado: Adicione uma propriedade sensores que seja um objeto com temperatura e distancia
  • JSON: Converta o objeto para JSON com JSON.stringify e exiba

📋 O que entregar

  • Objeto criado
  • Acesso por ponto/colchetes
  • Objeto aninhado
  • Conversão JSON

⭐ Critérios

  • Usou ponto e colchetes
  • Criou objeto aninhado
  • Usou JSON.stringify
