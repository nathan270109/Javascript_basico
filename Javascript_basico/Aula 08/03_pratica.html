Passo 1 - Crie função que receba dois números e retorne soma

▼
Instrução: Crie uma função chamada somar que receba dois parâmetros (a, b) e retorne a soma deles. Chame a função com 5 e 3.
Dica: Use function nome(a, b) {
return a + b; }
function somar(a, b) {
return a + b;
}
let resultado = somar(5, 3);
console.log(resultado);

Passo 2 - Escreva função que verifique se um número é positivo

▼
Instrução: Crie uma função isPositivo que receba um número e retorne true se for maior que 0, ou false caso contrário.
Dica: Use return n > 0 para comparar.
function isPositivo(n) {
return n > 0;
}
console.log(isPositivo(10));
console.log(isPositivo(-3));

Passo 3 - Use arrow function para multiplicar

▼
Instrução: Crie uma arrow function que receba dois números e retorne a multiplicação deles. Armazene em uma variável chamada multiplicar.
Dica: let multiplicar = (a, b) => a * b;
let multiplicar = (a, b) => a * b;
console.log(multiplicar(4, 3));

Passo 4 - Crie um callback para filtrar array

▼
Instrução: Crie um array de números e use filter() com uma função callback para selecionar os maiores que 3.
Dica: filter recebe uma função que retorna true/false.
let nums = [1, 2, 3, 4, 5];
let maiores = nums.filter(function(n) {
return n > 3;
});
console.log(maiores);

Passo 5 - Implemente função com parâmetro default

▼
Instrução: Crie uma função cumprimentar que receba um nome e exiba "Olá, [nome]!". Se nenhum nome for passado, use "Visitante" como padrão.
Dica: function cumprimentar(nome = "Visitante") { }
function cumprimentar(nome = "Visitante") {
console.log("Olá, " + nome + "!");
}
cumprimentar("Ana");
cumprimentar();

Passo 6 - Chame uma função dentro de outra

▼
Instrução: Crie duas funções: quadrado(n) e somaQuadrados(a, b). somaQuadrados deve chamar quadrado internamente.
Dica: Uma função chama a outra pelo nome.
function quadrado(n) {
return n * n;
}
function somaQuadrados(a, b) {
return quadrado(a) + quadrado(b);
}
console.log(somaQuadrados(3, 4));
