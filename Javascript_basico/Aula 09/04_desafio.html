Missão DOM — selecionar, modificar e interagir

Você é um Robô em uma missão de interação com páginas web. Precisa manipular elementos do DOM para tornar a página dinâmica. Complete os 4 requisitos para ativar o módulo de interface.

📝 Requisitos — clique para marcar

⬜ 1. Selecionar: Use getElementById para selecionar um elemento e alterar seu texto
⬜ 2. Modificar estilo: Use classList.add() para adicionar uma classe a um elemento
⬜ 3. Evento de clique: Adicione um evento click a um botão que exiba um alerta
⬜ 4. Criar elemento: Use createElement e appendChild para adicionar um novo parágrafo ao body

📋 O que entregar

  • Elemento selecionado
  • Estilo modificado
  • Evento de clique
  • Elemento criado

⭐ Critérios

  • Usou seletores corretamente
  • Modificou conteúdo/estilo
  • Implementou evento
  • Criou elementos
