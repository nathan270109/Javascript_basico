Passo 1 - Selecione um elemento e mude seu texto ▶

Instrução: Use getElementById para selecionar um elemento com id "mensagem" e altere seu texto para "Olá, DOM!".

Dica: document.getElementById("id").textContent = "texto";

let el = document.getElementById("mensagem"); 
el.textContent = "Olá, DOM!";
Passo 2 - Adicione uma classe a um elemento ▶

Instrução: Selecione um elemento com querySelector e adicione a classe "destaque" usando classList.add.

Dica: elemento.classList.add("nomeDaClasse");

let el = document.querySelector("p"); 
el.classList.add("destaque");
Passo 3 - Crie um botão que muda cor de fundo ▶

Instrução: Crie um botão no HTML. Ao clicar, a cor de fundo da página muda para uma cor aleatória.

Dica: Use addEventListener('click') e document.body.style.backgroundColor.

let btn = document.getElementById("btnCor"); 
btn.addEventListener("click", function() { 
document.body.style.backgroundColor = "lightblue";
 });
Passo 4 - Capture tecla pressionada ▶

Instrução: Adicione um evento keydown ao documento que exiba no console a tecla pressionada.

Dica: document.addEventListener('keydown', function(e) { console.log(e.key); });

document.addEventListener("keydown", function(event) { 
console.log("Tecla: " + event.key); 
});
Passo 5 - Crie uma lista dinâmica com input ▶

Instrução: Crie um input e um botão. Ao digitar um texto e clicar, o item é adicionado a uma lista (ul).

Dica: createElement('li'), appendChild, value do input.

let input = document.getElementById("itemInput"); 
let btn = document.getElementById("addBtn"); 
let lista = document.getElementById("lista"); 
btn.addEventListener("click", function() { 
let li = document.createElement("li"); 
li.textContent = input.value; 
lista.appendChild(li); 
input.value = ""; 
});
Passo 6 - Implemente delegação de evento ▶

Instrução: Em uma lista (ul), adicione um evento no pai que detecte qual li foi clicada e exiba seu texto.

Dica: Use e.target.tagName para verificar se clicou em um li.

let lista = document.getElementById("lista"); 
lista.addEventListener("click", function(e) { 
if (e.target.tagName === "LI") { 
console.log("Clicou em: " + e.target.textContent);
 }
 });
