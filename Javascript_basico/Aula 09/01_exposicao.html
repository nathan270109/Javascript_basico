O que é o DOM?

DOM (Document Object Model) é a representação da página HTML como uma árvore de objetos. Cada tag é um nó que pode ser acessado e modificado pelo JavaScript. O DOM transforma o HTML em uma estrutura que o JS entende.

Selecionar elementos: getElementById, querySelector

getElementById('id') seleciona um elemento pelo ID. querySelector('.classe') seleciona o primeiro elemento que corresponde ao seletor CSS.

let titulo = document.getElementById('titulo'); let paragrafo = document.querySelector('p');

Modificar conteúdo: textContent, innerHTML

textContent altera apenas o texto. innerHTML altera o HTML interno (pode incluir tags).

elemento.textContent = "Novo texto"; elemento.innerHTML = "<strong>Negrito</strong>";

Modificar atributos e classes

setAttribute() altera atributos. classList.add() e classList.remove() manipulam classes CSS. classList.toggle() alterna uma classe.

elemento.classList.add('destaque'); elemento.setAttribute('src', 'imagem.jpg');

Criar e remover elementos

createElement() cria novo elemento, appendChild() adiciona ao DOM, remove() ou removeChild() remove.

let novoP = document.createElement('p'); novoP.textContent = "Olá"; document.body.appendChild(novoP);

Eventos: click, mouseover, keydown

Eventos são ações que disparam funções. addEventListener() escuta eventos. click (clique), mouseover (passar mouse), keydown (tecla pressionada).

botao.addEventListener('click', function() { alert('Clicou!'); });

Evento load e DOMContentLoaded

DOMContentLoaded dispara quando o HTML foi carregado (antes de imagens). load espera tudo carregar (imagens, CSS).

document.addEventListener('DOMContentLoaded', function() { console.log('Pronto!'); });

Delegação de eventos

Em vez de adicionar evento a cada elemento, adicionamos ao pai e usamos target para identificar qual filho foi clicado. Essencial para listas dinâmicas.

lista.addEventListener('click', function(e) { if (e.target.tagName === 'LI') { console.log(e.target.textContent); } });
